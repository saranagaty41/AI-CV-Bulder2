# **App Name**: ResumeCraft AI

## Core Features:

- CV Generation: Generates ATS-friendly CV content based on user input and selected templates.
- Template Selection: Provides 2 switchable formal CV templates with live previews.
- PDF Download: Allows users to download their generated CV as a PDF file.
- ATS Optimization Tool: An AI-powered tool to refine CV content for better readability by Applicant Tracking Systems (ATS).
- Supabase Integration: Uses Supabase for user management and data storage (CV templates, metadata, user profiles).
- User Authentication: Implements user authentication using Supabase Auth.
- CV Template Storage: Saves and retrieves user-created CV templates from the Supabase database.

## Style Guidelines:

- Primary color: Gradient from green to turquoise for a modern and refreshing feel.
- Background color: Light gray (#F0F4F8) to ensure readability and a clean interface.
- Accent color: Teal (#45B3A6) for CTAs and important highlights, creating a modern feel.
- Body text font: 'Inter' sans-serif for a modern, clean and readable design.
- Headline font: 'Space Grotesk' sans-serif for headings, to add a hint of futuristic and technological feel to the presentation.
- Use clean, professional icons to represent different sections of the CV and user actions.
- Maintain a clean, well-structured layout with clear sections to guide the user through the CV creation process.
- Incorporate subtle transitions and animations for a smooth and engaging user experience.
